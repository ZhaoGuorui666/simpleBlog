##### 学习git提交流程 <http://www.bootcss.com/p/git-guide/>

1. git add *		//将改动添加到缓存区
2. git commit -m "代码提交信息"             //提交改动,但是还没到远端仓库
3. git push origin master              //推送改动到远端仓库,可以把master换成任何分支.
